Commands for working with graph-tools in Docker:

1) Show all currently running containers:

docker ps -a

2) Remove all currently stopped containers:

docker rm (docker ps -a -f status=exited -q) 
 
2.5) Stop all containers:

docker stop (docker ps -a -q)

3) Link a graph-tools python container called my_graph_tools1 to the local directory, where the name inside the container for the local directory is /mydata:

docker run -it --name my_graph_tools1 -v /Users/carter/Documents/School/Fall_2019/Research/:/mydata tiagopeixoto/graph-tool ipython

4) Connect a new terminal to an already existing docker container. First do docker ps -a to get the container ID. 

docker exec -it <CONTAINER ID> bash
